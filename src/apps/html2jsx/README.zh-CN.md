##### JSX

JSX 是 JavaScript XML,是 React 提供的 Syntax Sugar, 能让我们可以在 JS 中写 html 标记语言。

##### 支持属性

目前支持以下属性

-   class
-   style
-   contentEditable

不支持部分 Html 属性，请手动改为小驼峰命名法，所有操作均在本地执行，不消耗数据流量
